Getting Started

Welcome to Prognostika - The 1st AI-Powered Hard Drive Health Prediction System We would like to thank you for choosing our template - sinco.

It’s built with React js. Sinco is a truly unique, modern, beautiful and responsive React template that fits for all kinds of Business website, Data cience, Machine Learning, IT website, SAAS website, SEO, Agency, IT Services.This design is very creative and unique, and also very easy to customize and use further more, Sinco is fully 100% responsive. It will display your content nicely on Desktop, Laptop, iPad, iPhone, Android Mobile, or tablets and other screens.

Pre Requirements

There are system requirements in order to install and setup this template and its components properly. Make sure that you already installed in your machine node & npm.

Recommended node and npm 

                        
                            node install
                            npm install

Install Npm Packages

NPM : You must have to npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies. 

                        
                            npm install / npm install --legacy-peer-deps

Development server

Run npm run start for a dev server. Navigate to http://localhost:3000/ . The app will automatically reload if you change any of the source files. 

                        
                            npm run start
